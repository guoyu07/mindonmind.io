[Notes: Analysis 2.3] 基础拓扑：列紧、紧致与连通性

数学分析教程 下 p.84 - p.90

这部分主要介绍了欧式空间中的另一些拓扑属性及其相互之间的关系。

首先是列紧 (sequentially compactness)和紧致 (compactness)的概念。如果 \(\boldsymbol{R}^n\) 子集 \(E\) 中的任一点列都有一子列收敛于 \(E\) 中的某点，那么称 \(E\) 是列紧的。而如果从 \(E\) 的任意一个开覆盖中能选取出有限个开集，它们仍能构成 \(E\) 的开覆盖，就称 \(E\) 为紧致的。

关于列紧和紧致，有一个重要的定理，即它们的充分必要条件均是 \(E\) 为有界闭集。首先来考察列紧。我们知道闭集有一个重要的判断方法，即它的导集属于自身（或者说它的凝聚点属于自己，它的任意收敛点列收敛至自己的一点）。这样，列紧集显然是闭集，再有它的每个点列中均有收敛子列知道它必须是有界的。相反，由 Bolzano-Weierstrass 定理可知，有界闭集中的任一点列均可找出一收敛子列，所以有界闭集也是列紧的。

再看紧致性，对于一个紧致集 \(E\) ，构造一系列以 \(\boldsymbol{0}\) 为球心，半径为 \(1,2,3,\dots\) 的球它们是 \(E\) 的开覆盖，从这当中必能选取出有限个球覆盖 \(E\)。那么这些球中半径最大的那个能包含 \(E\)，故 \(E\) 有界。再考察 \(E^c\)，从中选取一点 \(\boldsymbol{q}\)，再以 \(E\) 中每一点为球心构造半径为 \(r_1, r_2, \dots, r_q\) 的球 \(\{B_i\}\) ，使每个球都与 \(B_{r_i}(\boldsymbol{q})\) 不相交。\(\{B_i\}\) 是开覆盖，从中选取出有限个球覆盖 \(E\)，并求得最小的半径 \(r_m\)，那么 \(B_{r_m}(\boldsymbol{a})\) 不与这些球中的任意一个相交，也因此不与 \(E\) 相交，故 \(B_{r_m}(\boldsymbol{a}) \subset E^c\)，即 \(E^c\) 是开集。相反，如果 \(E\) 是有界闭集可以构造一列直径递减的长方体列 \(\{K_i\}\)，使得 \({\rm diam} K_i = \frac{1}{2} {\rm diam} K_{i-1}\)，且其中的每一个均无法被某个开覆盖中的有限个覆盖，由闭集套定理，知 \(\cap B_i\) 中存在唯一一点，该点必存在于上述开覆盖中的某一个开集，因此该点的某近邻 \(\xi\) 也属于该开集，有必定存在某一个 \(m\) 使得 \(K_m \subset \xi\)，从而 \(K_m\) 属于该开集，导致矛盾。

因此，列紧集、紧致集与有界闭集是相互等价的三个概念。而在证明过程中，Bolzano-Weierstrass 和闭集套定理起到了重要作用。

下面再看另一个重要的拓扑性质——连通性 (connectedness)。所谓连通，即指将 \(E\) 分成两个不相交的非空子集时，至少有一个含有另一个的凝聚点。而我们知道，凝聚点反应了一种稠密性，即在它的周围总有无限多个收敛到它的点。如果一个集合含有另一个的凝聚点，也就是说这两个集合见不存在一段「空隙」，其中不含任一集合的点。

\(\boldsymbol{R}^n\) 中连通的开集称为区域，而连通的闭集称为闭区域。而在 \(\boldsymbol{R}\) 上，我们有：一个集合连通当且仅当它是一个区间(p88 定理 13.16)。所以，区域和区间是相对的概念，它们都是指连通集。

最后，关于连通集有一个判定定理，即道路连通集必是连通集。这里的道路连通是指，在集合中任意两点 \(\boldsymbol{p}\) 和 \(\boldsymbol{q}\) 之间都存在一条连续的曲线将它们相连。该定理的证明主要是利用连续曲线的参数表达形式，将 \(\boldsymbol{R}^n\) 中的连通性转化为 \(\boldsymbol{R}^n\) 中区间的连通性来证明。

由上面这条定理可知，任意凸集均是连通集。
